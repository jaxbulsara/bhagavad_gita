# Srimad Bhagavad Gita

This is a LaTeX rendition of the Srimad Bhagavad Gita with translations by
Swami Chinmayananda from his commentary, The Holy Geeta. It is set in a small
page format, making it suitable as a companion for chanting.

## Building

Building the pdf can be done via the `Makefile`.

```
$ make
```

This will generate the pdf at `./build/bhagavad_gita.pdf`

To remove the build files, run:

```
$ make clean
```
